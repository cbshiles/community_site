HTML tags are enclosed in angle brackets < >. There are opening and closing tags. The closing tag ends the content inside the tag, and has a slash before the tag name. Sometimes there are opening tags that don't have closing tags. This means that that type of tag can't have more HTML inside of it as content.

Example:
<p>Paragraph 1.</p> <p style="color:red;">Paragraph 2</p>

Paragraph 1.

Paragraph 2

Inside of these brackets are the tag name, and the opening tag can also have properties for the tag.

Example:
<a href="http://google.com">Link to Google!</a>
Link to Google!

The image tag is an example of a tag that doesn't come in a pair. This is because you're not allowed to put more content inside of the picture!

Example:
<img src="http://www.webstaurantstore.com/images/products/extra_large/89188/726588.jpg" width="200" height="200">

You can leave off the width and height if you want the image to maintain its original size.

An iframe tag is how you embed a video or another website. Youtube gives you this code if you click Share and then Embed right under the video.

Example:
<iframe width="560" height="315" src="https://www.youtube.com/embed/d4ZTRKh91us" frameborder="0" allowfullscreen></iframe>
